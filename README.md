# ML_GDP_clustering_and_correlation
Assignment in unsupervised learning for an introductory course in machine learning. Performs clustering of countries based on GDP per capita, finds the countries that have changed clusters over the years, and prints out correlations between GDP and some other variables for those countires (currently only variables that have complete timeseries 1960-2014).

To use the program, You need to downlad the dataset from here: https://data.worldbank.org/topic/climate-change?end=2014&start=1960 and place the API_19_DS2_en_csv_v2_10400593.csv file in the same directory as the scripts.
